# BMWK Policy Content Scraper

Lightweight Python scraper for extracting structured policy and event content from German Federal Ministry (BMWK/BMWE) webpages.

## Features
- Scrapes EU government websites using BeautifulSoup
- Extracts title, date, and full policy content
- Outputs clean CSV for downstream analysis

## Tech Stack
- Python
- requests
- BeautifulSoup
- pandas

## Example Use Case
Extract structured policy text for:
- policy analysis
- NLP / LLM pipelines
- comparative governance research

## Notes
Designed for academic and non-commercial research.
